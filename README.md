# 时间戳识别工具 - Chrome/Edge 浏览器扩展

一个简洁优雅的浏览器扩展，可以双击网页中的毫秒级时间戳自动识别并转换为可读的日期时间格式。

## 功能特性

✨ **双击识别**: 在任何网页中双击时间戳数字即可自动识别并转换  
🎨 **精美界面**: 渐变色设计，现代化的 UI 风格  
📋 **一键复制**: 点击任意时间格式即可复制到剪贴板  
🕐 **多种格式**: 提供标准格式、本地时间、ISO 8601、日期、时间等多种格式  
⚡ **轻量快速**: 无需后台权限，不影响浏览器性能  

## 安装方法

### Chrome 浏览器

1. 打开 Chrome 浏览器，在地址栏输入 `chrome://extensions/`
2. 开启右上角的"开发者模式"
3. 点击"加载已解压的扩展程序"
4. 选择本项目文件夹（包含 `manifest.json` 的文件夹）
5. 安装完成！

### Edge 浏览器

1. 打开 Edge 浏览器，在地址栏输入 `edge://extensions/`
2. 开启左下角的"开发人员模式"
3. 点击"加载解压缩的扩展"
4. 选择本项目文件夹（包含 `manifest.json` 的文件夹）
5. 安装完成！

## 使用方法

1. 安装扩展后，无需任何配置即可使用
2. 在任意网页中，双击一个毫秒级时间戳数字（例如：1699000000000）
3. 会弹出一个精美的悬浮窗，显示转换后的多种时间格式
4. 点击任意格式即可复制到剪贴板
5. 点击关闭按钮或页面其他区域可隐藏悬浮窗

### 迷你时间戳转换器（工具栏弹窗）

1. 安装扩展后，点击浏览器工具栏图标即可打开迷你转换器（已集成为弹窗）
2. 支持「时间 → 毫秒时间戳」与「毫秒时间戳 → 时间」双向转换
3. 默认使用浏览器本地时区进行解析与格式化
4. 提供「填入当前时间」与「示例时间戳」快捷按钮

## 时间戳范围

- 支持毫秒级时间戳（13位数字）
- 时间范围：1970-01-01 至 2100-12-31
- 示例时间戳：
  - `1699000000000` → 2023-11-03 16:53:20.000
  - `1730678400000` → 2024-11-04 00:00:00.000

## 显示格式

扩展会显示以下5种时间格式：

1. **标准格式**: YYYY-MM-DD HH:mm:ss.SSS
2. **本地时间**: 根据系统设置显示
3. **ISO 8601**: 国际标准时间格式
4. **日期**: YYYY-MM-DD
5. **时间**: HH:mm:ss

## 注意事项

- 该扩展仅在网页加载完成后生效
- 需要双击选中完整的时间戳数字
- 时间戳必须是整数且在有效范围内
- 点击提示框中的时间格式可复制到剪贴板

## 开发说明

如需修改或扩展功能：

1. 编辑 `content.js` 修改核心逻辑
2. 编辑 `styles.css` 自定义样式
3. 修改后在扩展管理页面点击"重新加载"按钮即可看到效果
